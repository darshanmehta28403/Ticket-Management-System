# 🎫 Ticket Management System

A full-stack Ticket Management System built using **Node.js, Express.js, React.js, PostgreSQL, Prisma ORM, Redis, and WebSockets**. The application enables teams to efficiently manage the complete ticket lifecycle with secure authentication, role-based authorization, real-time updates, and a scalable backend architecture.

## 🚀 Features

### Authentication & Authorization
- JWT Authentication
- Role-Based Access Control (RBAC)
- Protected Routes
- Secure Password Handling

### Ticket Management
- Create Tickets
- Update Ticket Status
- Assign Tickets
- Set Priority
- Track Ticket Progress
- Search & Filter Tickets
- Pagination

### Real-Time Updates
- Instant Status Synchronization

### Performance
- Optimized REST APIs
- Server-side Pagination
- Efficient Database Queries

### API
- RESTful Architecture
- Request Validation
- Centralized Error Handling
- Modular Backend Structure
